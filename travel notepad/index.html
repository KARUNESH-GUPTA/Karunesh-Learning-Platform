About This Project

VoyageLog is a beautiful, frontend-only travel journal where users can record trips, add photos, and save memories locally. This project demonstrates clean UI design, offline-first data persistence, and image-rich storytelling.

Motivation: "Create beautiful software that stores memories securely and looks great on every device."
